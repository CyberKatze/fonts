## How use it in Linux
- Copy fonts in `~/.fonts` or clone directly
```bash
git clone https://github.com/cyberkatze/fonts ~/.fonts
```
- `fc-cache` to cache fonts
- `fc-list` use `grep` to make sure your font exist
